Projects:

SeenIt!

SeenIt! is a website (on my local network) running on a Raspberry Pi Zero that helps users remember which episode of a television show they last watched. Users are able select their list of shows, edit the season and episode of current shows as well as add new shows to the list. This is an ongoing project and the features are constantly being re-evaluated and updated.

Languages and technologies involved:

  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL
  • GitHub

Parental Bank Consortium

Parental Bank Consortium is a small web application that was developed as a final project for my "Web Development with .Net" course. This project is a prototype of a potential backing website that would allow parents to monitor their child's spending patters as well as help make payments and supply funding for certain "Wish List" items the children may have.

Languages and technologies involved:
  • C#
  • .Net
  • VisualStudio
  • GitHub
